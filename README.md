# AiObsydian 🚀

An advanced, local AI Agent Orchestrator that connects **Ollama** LLMs with your **Obsidian Vault** using semantic memory (LanceDB) and a Telegram interface.

### ✨ Features (In Development)
* **Local LLM Integration:** Powered by Ollama (Hermes 3, Qwen 3.5).
* **Semantic Memory:** Uses LanceDB for RAG (Retrieval-Augmented Generation) over Obsidian notes.
* **Smart VRAM Scheduler:** Optimizes local GPU usage.
* **Telegram Interface:** Full agent control via Telegram bot with message batching.

### 🛠️ Tech Stack
* Python 3.14
* Aiogram (Telegram Bot API)
* LanceDB & SQLite
* Asyncio Task Queue
